# Imagen Express

## Usage (WINDOWS)

Inside ImagenExpress folder and then over websockets folder, run:

``` bash
npm install 
```

In ImagenExpress folder:

``` bash
npm start
```

In  websockets folder:

``` bash
node index.js
```



## Usage (LINUX)

On the ImagenExpress folder run:
 
-npm run install 
-npm run start

On the websockets folder:

-node index.js
